0: Instalación

La página oficial de JQuery ofrece varias opciones de instalación. En este seminario hemos usado un módulo de node siguendo las siguientes instrucciones:

npm install
python -m http.server

Ten en cuenta que puedes usar yarn en lugar de npm sin problemas y que la instalación se hace automática porque se proporciona un fichero package.json, por lo que no necesitas inicializar el paquete.

2: Eventos

Los eventos son muy útiles para ejecutar acciones cuando el usuario realiza una acción en la página. Prueba a descomentar algunos eventos en el script, pulsar el botón y ver qué ocurre:

Bloque animado

Una cosa muy interesante es que muchas funcionalidades son como los Sith: necesitan dos funciones que las gestionen. Por ejemplo, slideUp y slideDown son dos funciones opuestas. Para estos casos, siempre hay una versión toggle (como slideToggle) que comprueba el estado del objeto y ejecuta la función que lo modifique al opuesto.
